An implementation and domain specific language for the Carneades
argumentation model. See Haskell Gets Argumentative in the 
Proceedings of Symposium on Trends in Functional Programming 
(TFP 2012) by Bas van Gijzel and Henrik Nilsson or the 
package's homepage: http://www.cs.nott.ac.uk/~bmv/CarneadesDSL/. 
Thanks to Stefan Sabev for providing initial code for the cyclicity check.
